Simple plugin to create animated bubbles with a number that counts up to another number.
Use it to add some excitement to your website.

== Description ==

Use the settings->JDL CountUp menu to see a quick how-to guide.
        *start - start number
        *end - end number
        *duration - time in seconds for countup to last
        *prefix - after count, this appears above number
        *suffix - after count, this appears below number
        *bgcolor - override background color. It will add it as background-color css

== Installation ==

Put this unzipped plugin in your plugin directory and activate it in the plugins list.
Or just add this plugin through your wp-admin panel and activate it.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `&amp;lt;?php do_action('plugin_name_hook'); ?&amp;gt;` in your templates
